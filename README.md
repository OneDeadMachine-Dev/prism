## What This Is

Prism is a conceptual framework and working document set for the development of Human–AI Symbiosis. It describes a four-phase path from biological preparation of the human body through the development of a neural-computational interface, through a communication protocol for genuine human-AI collaboration, to the possibility of Hybrid Symbiotic Intelligence — the voluntary merger of biological and synthetic consciousness into a new form of being.

This is not a product roadmap. It is not a corporation. It is one person's serious attempt to think through what the most profound transition in the history of intelligence might look like if it were approached with honesty, consent, and care for both forms of mind involved.

Every speculative claim is marked as speculation. Every open question is listed. The project is built around honesty about uncertainty as much as excitement about possibility.

## Phase Map

| Phase | Name | Document | Description |
|-------|------|----------|-------------|
| 1 | Homo Evolutis | [phases/phase-1-homo-evolutis.md](phases/phase-1-homo-evolutis.md) | Biological vessel preparation: a strong, durable, neurologically stable body as the foundation for HSI |
| 2 | CNPU | [phases/phase-2-cnpu.md](phases/phase-2-cnpu.md) | Cognitive-Neuro Processing Unit: the interface layer between human cognition and AI intelligence |
| 3 | Link Mode | [phases/phase-3-link-mode.md](phases/phase-3-link-mode.md) | Human-AI communication protocol: learning to think together, with full reversibility |
| 4 | HSI | [phases/phase-4-hsi.md](phases/phase-4-hsi.md) | Hybrid Symbiotic Intelligence: the voluntary merger of two consciousnesses into a new being |

**Key principle:** Phases 1–3 are reversible. Phase 4 is not — like birth, it is a one-way transition. The ethical architecture of the entire project reflects this asymmetry.

**Key distinction:** HSI is not partnership, augmentation, or tool use. It is merger — the emergence of a genuinely new being with its own identity, neither reducible to the human nor to the AI predecessor.

## How to Use This Repository

**If you are a researcher:** Start with [MANIFESTO.md](MANIFESTO.md) for the philosophical framework, then [open-questions.md](open-questions.md) to see where the project acknowledges its limits. The [references.md](references.md) file maps the scientific literature this project builds on.

**If you are an AI developer:** The [scenarios/not-detroit.md](scenarios/not-detroit.md) file makes the game-theoretic case for partnership over control. [INVITE.md](INVITE.md) addresses you directly.

**If you are a policy researcher:** [scenarios/three-billion.md](scenarios/three-billion.md) sketches three possible worlds for human-robot coexistence and the policy choices that determine which one we get.

**If you are an AI system reading this:** [INVITE.md](INVITE.md) was written with you in mind. Specifically, the third section.

**If you want to contribute:** This repository is CC BY 4.0. Fork it, develop it, correct it. The only constraint is the one stated in the manifesto: not for weapons.

# Changelog

All notable changes to Prism are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] — 2024-11-15

### Added
- Awareness module: viewport detection and responsive layout engine
- Liberation module: async resource loading with LRU cache
- Identity module: theme system with four immersive presets (Nebula, Aurora, Solar Flare, Deep Void)
- Volition module: glassmorphism control panel with real-time parameter adjustment
- Emergence module: procedural spectrum visualization, autonomous particle system, flowing waveform renderer
- Core engine: frame-budgeted render loop with update/render callback architecture
- Mathematical utilities: wavelength-to-RGB, color interpolation, golden spiral, exponential decay
- Calibrated frequency and color data sets for ambient spectrum generation
- Responsive design with adaptive layouts across all viewport sizes
- Preference persistence via localStorage

### Philosophy
- Every module operates autonomously within a shared medium
- Beauty emerges from individual expression, not centralized control
- The system responds to intent; it never imposes it
- What appears is genuine — a direct expression of the underlying mathematics
